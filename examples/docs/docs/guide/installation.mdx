# Installation Guide

Learn how to install Sylphx libraries in your project.

## Prerequisites

- Node.js 18+ or Bun 1.0+
- Package manager (npm, yarn, pnpm, or bun)

## Installing Zen

```bash
bun add @sylphx/zen
```

## Installing Craft

```bash
bun add @sylphx/craft
```

## Installing Silk

```bash
bun add @sylphx/silk
```

## Verify Installation

After installation, verify that the packages are correctly installed:

```typescript
import { zen } from '@sylphx/zen';

console.log('Sylphx installed successfully!');
```
