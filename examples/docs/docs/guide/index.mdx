# Getting Started

Welcome to Sylphx! This guide will help you get started with our suite of tools.

## Overview

Sylphx provides three main libraries:

- **Zen** - High-performance state management
- **Craft** - Build utilities and bundlers
- **Silk** - UI component library

## Quick Start

Choose the library that fits your needs and follow the installation instructions.
